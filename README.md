# Arkham AI Module Generator

> 一款基于 AI 的克苏鲁的呼唤 (CoC) TRPG 模组生成器

## 📖 简介

Arkham Architect 是一个 Web 应用，帮助守密人 (Keeper) 快速生成《克苏鲁的呼唤》跑团模组。通过集成 AI，自动创作模组设定、剧情分支、NPC 档案和调查场景。

## ✨ 功能特性

- **🎲 AI 灵感生成** - 自动构思时代背景、地点、反派和关键道具
- **💡 智能补全** - 支持先填写部分设定，AI 自动补全空白项
- **🕯️ 剧情分支推演** - 生成三种不同风格的调查员切入方向
- **🖋️ 模组自动书写** - 一键生成完整的模组真相、时间轴和决战场景
- **👥 NPC 档案生成** - 创建详细的角色信息，包含秘密设定
- **🔍 探索场景设计** - 生成调查区域、物品线索和触发事件
- **📖 模组排版预览** - 专业的双栏书本排版，支持导出 PDF
- **💾 本地存档系统** - 保存和读取创作进度
- **🎨 多主题切换** - 黄衣之王 / 黑色大丽花 / 拉莱耶 三种视觉风格
- **🔗 多 API 支持** - 支持 Gemini、OpenAI 及兼容格式（DeepSeek等）

## 🚀 快速开始

### 1. 获取 API Key

本工具支持多种 AI 提供商：

| 提供商 | 获取地址 |
|---------|----------|
| Google Gemini | [Google AI Studio](https://aistudio.google.com/app/apikey) |
| OpenAI | [OpenAI Platform](https://platform.openai.com/api-keys) |
| 其他OpenAI格式提供商平台 | - |

### 2. 启动应用

项目网址[https://kar4noshojo.github.io/arkham-generator/](https://kar4noshojo.github.io/arkham-generator/)

### 3. 配置 API

首次使用时，点击侧边栏的「密钥设置」：
1. 选择 API 类型（Gemini / OpenAI 格式）
2. 输入你的 API Key
3. 可选：自定义 Base URL 和模型名称

所有配置仅存储在本地浏览器中。

## 📝 使用流程

1. **灵感源泉** - 可先填写部分设定（如时代、地点），点击「铭刻灵感」让 AI 补全剩余空白项
2. **叙事走向** - 点击「推演命运」获取三种剧情分支，选择一个方向
3. **模组全貌** - 点击「自动书写」生成完整的模组内容
4. **人物档案** - 切换到角色档案页，生成 NPC 信息
5. **探索场景** - 切换到场景页，生成调查区域
6. **预览导出** - 查看排版效果，使用浏览器打印功能导出 PDF

## ⚠️ 注意事项

- API Key 和配置仅存储在浏览器本地存储中，不会上传至任何服务器
- 使用 OpenAI 兼容格式时，可自定义 Base URL 连接任意兼容服务
- 建议定期使用「封存」功能保存创作进度
